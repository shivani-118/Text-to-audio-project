# Audio to Text Converter

This project is a web application that converts speech (audio) into text using the Whisper model.

The user uploads an audio file, and the system converts the speech into written text.

## Technologies Used
- Python
- Flask
- OpenAI Whisper Model
- FFmpeg
- HTML
- CSS

## Features
- Convert audio speech into text
- Simple and user-friendly web interface
- Fast transcription using Whisper model

## Requirements
- Python
- Whisper library
- FFmpeg

## How to Run the Project

1. Install required libraries
pip install openai-whisper

2. Install FFmpeg on your system

3. Run the application
python app.py

4. Open browser and go to
http://127.0.0.1:5000

## Project Output
The user uploads an audio file and the system converts the speech into text using the Whisper model.
